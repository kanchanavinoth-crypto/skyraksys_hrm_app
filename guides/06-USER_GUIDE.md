## ðŸ‘¨â€ðŸ’¼ Admin Features

### User Management

**Create New User**:
1. Navigate to **Users** > **Add User**
2. Fill in user details:
   - Email (required, unique)
   - Role (admin/hr/manager/employee)
   - Temporary password
3. Click **Create User**
4. User will receive welcome email with password reset link

**Manage Existing Users**:
- View all users
- Edit user roles
- Deactivate users
- Reset passwords
- View user activity logs

### Employee Management

**Add New Employee**:
1. Navigate to **Employees** > **Add Employee**
2. Fill required fields:
   - **Basic Info**: Employee ID, Name, Email, Phone
   - **Personal Info**: DOB, Gender, Address
   - **Employment**: Department, Position, Manager, Hire Date
   - **Bank Details**: Account Number, IFSC Code
   - **Salary**: Basic Salary
3. Click **Save**

**Import Multiple Employees**:
1. Go to **Employees** > **Import**
2. Download CSV template
3. Fill template with employee data
4. Upload completed CSV
5. Review and confirm import

### Department & Position Management

**Manage Departments**:
- Add/Edit/Delete departments
- Assign department heads
- View department statistics

**Manage Positions**:
- Create job positions
- Set salary ranges
- Link to departments

### System Configuration

**Leave Types**:
- Configure leave types (Casual, Sick, Earned)
- Set allocation per year
- Enable/disable leave types

**Payroll Settings**:
- Configure PF/ESI percentages
- Set professional tax slabs
- Define payslip templates

### Reports & Analytics

Available reports:
- Employee directory
- Department-wise headcount
- Attendance summary
- Leave reports
- Payroll summary
- Audit logs

## ðŸ‘©â€ðŸ’¼ HR Features

### Employee Lifecycle Management

**Onboarding**:
1. Create employee record
2. Assign to department and manager
3. Set up leave balance
4. Generate login credentials

**Salary Management**:
- View all salary structures
- Update salary components
- Process annual increments

### Leave Management

**View Leave Requests**:
1. Go to **Leaves** > **Requests**
2. Filter by status (Pending/Approved/Rejected)
3. View employee details and leave balance

**Approve/Reject Leaves**:
1. Click on leave request
2. Review details and dates
3. Add comments (optional)
4. Click **Approve** or **Reject**

**Manage Leave Balances**:
- View all employee balances
- Adjust balances (carry forward, manual adjustment)
- Generate leave reports

### Payroll Processing

**Generate Monthly Payroll**:
1. Go to **Payroll** > **Generate**
2. Select month and year
3. Select employees (or all)
4. Click **Generate Payroll**
5. System calculates:
   - Gross salary
   - Deductions (PF, Tax)
   - Net salary
6. Review and approve

**Generate Payslips**:
1. After payroll approval
2. Go to **Payroll** > **Payslips**
3. Select month
4. Click **Generate All Payslips**
5. Download or email to employees

### Timesheet Review

- View all submitted timesheets
- Approve/reject timesheets
- Export timesheet data
- Generate billable hours report

## ðŸ‘¨â€ðŸ’» Employee Features

### Profile Management

**Update Profile**:
1. Click profile icon > **My Profile**
2. Update editable fields:
   - Phone number
   - Address
   - Emergency contact
   - Bank details
3. Click **Save Changes**

**Change Password**:
1. Go to **Profile** > **Security**
2. Enter current password
3. Enter new password (twice)
4. Click **Update Password**

### Timesheet Management

**Submit Weekly Timesheet**:
1. Navigate to **Timesheets**
2. Click **Add Timesheet**
3. Select:
   - Week start date
   - Project
   - Task
4. Enter hours for each day (Monday-Sunday)
5. Add description/notes
6. Click **Save as Draft** or **Submit**

**Edit Draft Timesheet**:
- Drafts can be edited anytime
- Submitted timesheets cannot be edited

**View Timesheet History**:
- Filter by date range
- View approval status
- Download timesheet report

### Leave Management

**Apply for Leave**:
1. Go to **Leaves** > **Apply Leave**
2. Select leave type
3. Choose start and end dates
4. For half-day: Select "Half Day" and choose period
5. Enter reason
6. Click **Submit**

**Check Leave Balance**:
1. Navigate to **Leaves** > **My Balance**
2. View available balance for each leave type
3. See used and pending leaves

**View Leave History**:
- Past leave requests
- Approval status
- Comments from approver

### Payslips

**View Payslips**:
1. Go to **Payroll** > **My Payslips**
2. Select month and year
3. View detailed breakdown:
   - Earnings (Basic, HRA, Allowances)
   - Deductions (PF, Tax)
   - Net salary
4. Download PDF

**Yearly Summary**:
- View annual earnings
- Tax deducted (Form 16)
- Download yearly statement

## â“ FAQ

**Q: How do I reset my password?**  
A: Click "Forgot Password" on login page, enter email, and follow reset link.

**Q: Can I edit a submitted timesheet?**  
A: No, only draft timesheets can be edited. Contact your manager for submitted timesheets.

**Q: How far in advance can I apply for leave?**  
A: You can apply up to 3 months in advance.

**Q: When will I receive my payslip?**  
A: Payslips are generated by 5th of each month for the previous month.

**Q: Who approves my timesheets and leaves?**  
A: Your direct manager (as assigned in your profile) approves both.

**Q: Can I download my annual salary statement?**  
A: Yes, go to Payroll > Yearly Summary > Download.
