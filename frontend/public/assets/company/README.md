# Company Assets

This directory contains company branding assets for payslips and other documents.

## Required Files

1. **logo.png** - Company logo (recommended size: 200x50px)
2. **signature.png** - Authorized signature image (recommended size: 200x80px)

## Instructions

1. Add your company logo as `logo.png`
2. Add the authorized signature as `signature.png`
3. Both files should be in PNG format for best quality
4. Ensure the files are properly sized for good display in payslips

## Note

The payslip template will gracefully handle missing images by hiding them if they fail to load.
